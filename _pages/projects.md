## AI Research Projects

---

### WindyAI: Planning & Optimization System

**Problem:** Design an intelligent planning and optimization system that integrates visual perception with decision-making algorithms.

**Technical Approach:** Leverages computer vision for scene understanding combined with AI planning algorithms and optimization techniques for task sequencing and resource allocation.

**Contribution:** Developed the computer vision pipeline for environment perception; implemented optimization algorithms for plan generation; integrated components into unified system architecture.

**Keywords:** AI Planning  Optimization  Computer Vision  Scene Understanding  Decision Systems

---

### MindMeetingAI: Multimodal Meeting Intelligence Platform

**Problem:** Automatically capture, process, and retrieve key insights from unstructured meeting audio and discussion content.

**Technical Approach:** End-to-end system combining Automatic Speech Recognition (ASR), Retrieval-Augmented Generation (RAG), and large language models (LLMs) for real-time transcription, context retrieval, and semantic analysis.

**Contribution:** Designed and implemented ASR pipeline integration; developed RAG retrieval system; engineered LLM-based query system for meeting insights extraction; evaluated end-to-end system performance.

**Keywords:** ASR  Retrieval-Augmented Generation  Large Language Models  Meeting Analysis  Multimodal Processing

---

### Vietnamese Facial Emotion Recognition System

**Problem:** Classify emotional states from facial expressions in Vietnamese subjects with high accuracy across diverse lighting and pose conditions.

**Technical Approach:** Deep learning-based computer vision system using convolutional neural networks for facial detection and emotion classification; data augmentation and transfer learning to handle dataset limitations.

**Contribution:** Curated Vietnamese emotion dataset; designed and trained emotion recognition model; implemented facial preprocessing pipeline; conducted cross-validation and comparative analysis.

**Keywords:** Facial Emotion Recognition  Computer Vision  Deep Learning  Convolutional Neural Networks  Transfer Learning

---

### Real-Time Streaming Server: RTSP/RTP Protocol Implementation

**Problem:** Design a low-latency, efficient video streaming server supporting industry-standard protocols for distributed AI processing.

**Technical Approach:** Server implementation supporting RTSP/RTP protocols with frame buffering, synchronization, and multi-client connection management for streaming to edge devices and processing nodes.

**Contribution:** Implemented core streaming protocol handlers; designed buffer management system; optimized latency through parallel frame processing; integrated with computer vision pipeline.

**Keywords:** RTSP  RTP  Streaming  Real-Time Processing  Low-Latency Systems  Network Protocols
